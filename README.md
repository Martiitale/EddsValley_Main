
# EDDSVALLEY

Mod de Stardew Valley que introduce a los personajes de la web-serie Eddsworld; Edd, Matt, Tord y Tom, asi como una nueva localizacion y eventos de corazones nuevos.


## Autores

- [@Martiitale](https://martiitale.carrd.co/)
- [@DaniPraivet](https://github.com/DaniPraivet)


[## Installation ](Installation.png)

PASOS DE INSTALACION:

1.-Descargar [Smapi](https://smapi.io/) (Ver instrucciones de instalacion.)

2.-Descargar [ContentPatcher](https://www.nexusmods.com/stardewvalley/mods/1915)(Ver instrucciones de instalacion.)

3.-Descargar archivo EddsValley.zip

4.-Descomprimir EddsValley.zip en la carpeta de mods creada por Smapi
